# Changelog

## [v1.3] - 2026-07-09

### Added
- **全文搜索**：首页新增搜索栏，支持实时过滤与智能提示（如「上个月的高光」「关于咖啡的记录」）
- **年度特刊**：新增杂志年度特别版，包含年度卷首语、关键词排名、四季回顾、年度统计与主编寄语
- **记忆盲盒**：新增独立页面，随机抽取一条已记录的瞬间，支持「再来一个」与返回
- **AI 情感分析**：集成 HuggingFace BERT 多语言情感分类器（Xenova/bert-base-multilingual-uncased-sentiment），替代原有的规则打标系统
- **首页入口重构**：Hero 区域新增 CTA 按钮组，支持搜索与盲盒两个入口，增加 Ghost Button 次要样式

### Removed
- 移除旧的 `generateTagsForImage` 基于文件名的标签生成函数
- 移除旧的 `analyzeTextForTags` 基于关键词匹配的文本标签函数

### Changed
- 更新 `.gitignore`，排除 `test-report.html` 临时文件

---

## [v1.2] - 2026-07-09

### Added
- **首页瀑布流剪报墙**：以 4 列网格替代旧的预览卡片布局，展示更多内容
- **首页统计栏**：顶部新增「瞬间 / 期杂志 / 陪伴天数」实时计数
- **杂志入口横幅**：新增 Issue Banner 作为杂志入口，展示主编寄语与期号预览
- **IndexedDB 图片存储**：图片迁移至 IndexedDB Blob 存储，提升读写性能
- **图片压缩**：导入导出时自动压缩图片（1200px / 0.7 质量），减小数据体积
- **开发约束文档**：新增 `DEVELOPER_NOTES.md`，记录文件范围与数据存储规范

### Changed
- 重构首页布局，移除旧的 `.home-preview` 和 `.home-features` 模块
- 导入导出功能增强，支持 Blob 图片数据的序列化与反序列化
- 更新 `.gitignore`，排除备份 JSON 文件

## [v1.1] - 2026-07-06

### Added

- **版权页 (Copyright Page)**：新增版权信息页，展示出版详情与版权声明

## [v1.0] - 2026-07-03

### Added
- **完整项目结构**：主页面 `index.html`、PRD 文档、创意提案与原型系统
- **杂志阅读器**：基于纯 CSS/JS 的自定义翻页动画（替代 StPageFlip 库）
- **四种交替内容页排版**：A（全幅图片+引言）、B（图文并排）、C（引言块+小图）、D（极简居中），每 4 页循环切换
- **日历视图**：集成 localStorage，动态读取并展示真实用户记录
- **多模态照片分析**：前端 Canvas 采样（60x60 像素），分析色调、饱和度、对比度与场景推断
- **剪报墙响应式布局**：移动端 2 列、≥768px 3 列、≥1024px 4 列
- **杂志页双栏排版**：≥768px 启用 column-rule 双栏布局
- **日期选择面板**：未来日期禁用，选中日期以赤陶色高亮
- **记录页动态文案**：根据选中日期自动切换「今天」与「那天」文案
